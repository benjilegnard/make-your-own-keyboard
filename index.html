Problèmes rencontrés

Le cé cédille

  • Existe pas en qwerty
  • Solution: utiliser le layout US intl with dead keys
  • Touches mortes: `+e=è
  • Mettre un altGR (right alt) sur son layer 0

Passer d'azerty en qwerty

  • Configurer 2 langues de saisie
  • Windows : shift + capslock
  • Linux (gnome) : super + space

Changer ses habitudes

  • Compliqué quand on tape depuis longtemps sur le même layout
  • Rien n'est insurmontable, il suffit de répéter les choses.
